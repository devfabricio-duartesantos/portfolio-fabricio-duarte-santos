# 🍽️ QRnaMesa - Personalização de QR Codes com IA

## 📝 Descrição do Projeto
O **QRnaMesa** é uma plataforma avançada para criação e estilização de QR Codes, desenvolvida para transformar a forma como estabelecimentos gerenciam seus cardápios e identidades digitais. O projeto expande o modelo *QR Code Styling*, integrando recursos de **Inteligência Artificial Generativa** e uma infraestrutura robusta de gerenciamento de projetos.

O diferencial do sistema reside na capacidade de unir design técnico com criatividade automatizada, permitindo que o usuário não apenas gere um código funcional, mas construa uma peça de branding completa com suporte de IA para criação de logotipos e personalização de interface.



## 🚀 Diferenciais Implementados
O sistema introduz inovações significativas em relação aos modelos convencionais:

*   **🌓 Alteração de Temas:** Interface adaptável com suporte nativo a **Modo Claro** e **Modo Escuro**, garantindo melhor usabilidade em diferentes ambientes de trabalho.
*   **🎨 Geração de Logo com IA:** Integração com modelos generativos (Gemini API) para criação de logotipos via prompts de texto ou opção de upload manual de imagens.
*   **📂 Gestão de Projetos (Cloud & Local):** Layout redesenhado focado no fluxo de trabalho, permitindo a criação de novos projetos, upload de designs existentes e consulta de histórico completo.
*   **🔐 Sistema de Autenticação:** Login integrado via **Google Auth** ou acesso como Visitante. Usuários autenticados desbloqueiam recursos premium como salvamento em nuvem e histórico persistente.

## 🛠️ Tecnologias Utilizadas
*   **Linguagem:** Python 3.10+
*   **IA Generativa:** Google Gemini Pro (via Google AI Studio)
*   **Frontend:** Streamlit / Web Technologies
*   **Biblioteca Base:** QR Code Styling
*   **Autenticação:** Google OAuth 2.0

## 📊 Resultados e Aprendizados
O desenvolvimento do QRnaMesa permitiu o aprofundamento em conceitos críticos de software:
*   **Persistência de Dados:** Implementação de lógica para salvamento em nuvem e histórico de projetos para usuários logados.
*   **Integração de APIs:** Experiência prática na conexão com a API do Gemini para gerar ativos visuais (logos) em tempo real.
*   **UI/UX Progressivo:** Criação de uma interface que diferencia a experiência entre usuários visitantes e usuários registrados.
